QUICK REFERENCE
═══════════════════════════════════════════════════════════════════════════════

🚀 To Start Trading:
  1. python test_data_fetcher.py          (verify installation)
  2. python fetch_and_backtest.py         (fetch data & backtest)
  3. python main.py config                (check bot settings)
  4. python main.py backtest              (run full backtest)
  5. python main.py paper-trading         (simulate trading)

📊 To Fetch Data:
  python fetch_and_backtest.py --pairs EURUSD GBPUSD USDJPY

⚡ To Use Cache (Instant):
  python fetch_and_backtest.py --skip-fetch

🧪 To Run Tests:
  python test_data_fetcher.py

  cat DATA_FETCHER_README.md
  cat DATA_FETCHER_GUIDE.md
